# BlackJack

This project exists 6 working files (*.py) and 1 text file for writing some information during the operation of the program:

  1. Cards.py, GameRound.py, Player.py - files with classes of cards, player and class for registration player and comparing game's result

  2. server.py - file with server's part to process data from the client and issue a result
  
  3. client.py - file with client's part to request data and output the result of the game, which comes from the server
  
  4. additional_functions.py - file with auxiliary functions for encoding and decoding information and writing it in json

*** HOW TO PLAY ***

To start the game you need to start the server (server.py), then start the client (client.py) and follow the instructions shown in the console
